ANEMOI

Vientos de Argentina

Fuentes de datos para nuestro trabajo

Trabajamos con datos observados en 46 estaciones meteorológicas del Servicio Meteorológico Nacional (SMN) de Argentina y del Instituto de Teconología Agropecuaria (INTA) y con datos estimados a partir de imagenes satelitales de POWERNASA.
A las estaciones meteorológicas que no cuentan con 24 datos diarios no se les realizó la marcha diaria del viento.
En los mapas climátologicos y en las marchas anuales se usaron los datos disponibles de todas las estaciones en las horas 0,6,12,18 UTC.
Para realizar las rosas de los vientos se usaron todos los datos disponibles.

Procesamiento de datos

Los datos fueron procesados con Sqlite3 y librerias de Python

Programación

Los gráficos interactivos fueron diseñados con librerias de Python y Heigcharts de JavaScript

Elaboración de mapas

Los mapas fueron diseñados con el software QGIS y el mapa interactivo con la libreria Folium de Python

Página Web

La página fue diseñada por medio del framework de Python Flask y la biblioteca multiplataforma Bootstrap

¿Por qué Anemoi?

En la mitología griega, los Anemoi eran dioses de los vientos, cada uno correspondía a un punto cardinal desde el cual venia su viento y estaban relacionados con las estaciones del año y los estados meteorológicos.
